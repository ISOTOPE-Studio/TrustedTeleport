# Trusted Teleport
方便的传送插件

## Download/下载:

https://github.com/ISOTOPE-Studio/TrustedTeleport/tree/master/files

## 命令:

/tt  <玩家名字> - 传送到被信任的玩家

/ttp add <玩家名字> - 添加玩家到信任列表

/ttp remove <玩家名字> - 将玩家从信任列表移除

/ttp list - 查看信任列表

/ttp about - 查看插件信息

## 权限:

TrustedTeleport.teleport - 允许使用 /tt 命令传送

TrustedTeleport.setting - 允许使用 /ttp 命令
